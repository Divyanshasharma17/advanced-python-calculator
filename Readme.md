# 🧮 Advanced Python Calculator (CLI + GUI)

## 🚀 Features
- Basic arithmetic operations (+, -, *, /)
- Scientific functions (sin, cos, tan, sqrt, log)
- Keyboard input support
- Calculation history
- Modern dark UI

## 🛠 Tech Stack
- Python
- Tkinter

## ▶️ How to Run

2. Run the file:
